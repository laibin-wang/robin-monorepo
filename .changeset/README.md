# Changesets

Hello and welcome! This folder has been automatically generated by `@changesets/cli`.

## What is Changesets?

Changesets is a tool to help manage versioning and changelogs for your packages.

## How to use it?

1. When you make changes to a package, run `pnpm changeset` to create a changeset
2. Answer the prompts about what packages have changed and the type of change (major/minor/patch)
3. Write a summary of the changes
4. Commit the generated changeset file along with your code changes
5. When ready to release, run `pnpm version-packages` to bump versions
6. Run `pnpm release` to publish to npm
