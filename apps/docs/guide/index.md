# 介绍

COTC Plugins 是一个包含多个高质量 JavaScript/TypeScript 组件的集合。

## 特性

- 🎯 **TypeScript 优先** - 完整的类型定义
- 📦 **Tree-shakable** - 支持按需引入
- 🚀 **轻量无依赖** - 不依赖任何第三方库
- 🎨 **易于定制** - 灵活的配置选项

## 组件列表

| 组件                                      | 描述         | 版本  |
| ----------------------------------------- | ------------ | ----- |
| [SignaturePad](/components/signature-pad) | 签名板组件   | 1.0.4 |
| [MonitorSDK](/components/monitor-sdk)     | 前端监控 SDK | 0.1.0 |
